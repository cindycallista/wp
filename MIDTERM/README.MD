## 網站功能
- **商品分類**：為便於瀏覽進行商品分類。
- **客製化**：允許使用者個性化訂單。
- **購物車管理**：添加帶有客製化細節的商品到購物車。
- **免費配送**：會員登入後享受免費配送服務。
- **支付**：整合安全的支付頁面。
- **訂位功能**：允許使用者預訂餐桌。
- **意見反饋**：包含用戶意見反饋/聯繫我們區域。
